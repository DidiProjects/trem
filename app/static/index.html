POST /pdf/convert-to-image Convert PDF to images (PNG, JPEG, TIFF)
‚ñº

Parameters

NameTypeDescription
filefilePDF file
formatstringFormat: png, jpeg, tiff (default: png)
dpiintegerResolution: 72-600 (default: 150)
pagesstringPages to convert (optional, e.g., 1-3,5)

Response

Single page: direct image | Multiple: ZIP with images

cURL Example

curl -X POST "/pdf/convert-to-image" \
                                -H "X-API-Key: your-api-key" \
                                -F "file=@document.pdf" \
                                -F "format=png" \
                                -F "dpi=150" \
                                --output images.zip
POST /pdf/convert-to-ofx Convert bank statement PDF to OFX
‚ñº

Parameters

NameTypeDescription
filefileBank statement in PDF
bank_idstringBank ID (4 digits)
account_idstringAccount number
account_typestringCHECKING, SAVINGS or CREDITCARD

Note

Extracts transactions from Brazilian statements (DD/MM/YYYY).
For specific formats, adjustments may be necessary.

cURL Example

curl -X POST "/pdf/convert-to-ofx" \
                                -H "X-API-Key: your-api-key" \
                                -F "file=@statement.pdf" \
                                -F "bank_id=0001" \
                                -F "account_id=12345" \
                                -F "account_type=CHECKING" \
                                --output statement.ofx
